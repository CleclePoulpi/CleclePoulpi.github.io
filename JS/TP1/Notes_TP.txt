        1er exo 
Doc 1 : déclaration de l'alerte dans le script et appel "en dur" dedans
Doc 2 : déclaration de l'alerte dans le script et appel dans le body au chargement de la page (mot clé "onload")
    onload = "fonction()" permet d'appeler la fonction après le chargement de la page, ce qui permet au navigateur d'attendre que tous
    les fochiers aient été chargés, et non pas au moment de l'exécution du script
Doc 3 : déclaration de l'alerte dans le script et appel dans le script également au chargement de la page
    window.onload = fonction permet également d'attendre le chargement de tous les fichiers avant d'exécuter la fonction
    /!\ pas de parenthèses !!
